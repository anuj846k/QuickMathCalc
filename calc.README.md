## Overview

Welcome to Quick Math Calculator! This simple calculator is designed to perform basic arithmetic operations, making it a handy tool for quick calculations.

### Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Clear input (C)
- Evaluate (=)
